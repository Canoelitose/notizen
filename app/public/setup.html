Mit deiner Datenbank verbinden

Diese App speichert deine Notizen in deiner eigenen PostgreSQL-Datenbank. Verbinde sie einmalig — danach geht's direkt los.

1 · Datenbank

Praktisch für gehostete/Cloud-Datenbanken oder einen Cloudflare-Tunnel (127.0.0.1).
Wird verlangt, weil der Server von aussen erreichbar ist.

2 · Dein Login

Wird beim ersten Mal angelegt. Existiert in der DB schon ein Konto, kannst du das hier leer lassen.
Noch keine Datenbank? Lokale DB einrichten

Du brauchst eine laufende PostgreSQL. Schnellste Wege:

Windows (ein Skript, richtet alles ein + Autostart):

powershell -ExecutionPolicy Bypass -File local\setup-local.ps1

Docker (DB + App in einem):

cp .env.docker.example .env  &&  docker compose up -d

Ein echtes Ein-Klick „lokale DB automatisch" direkt aus diesem Fenster kommt mit der Desktop-App (Electron) — dort kann die App selbst eine eingebettete PostgreSQL starten. Bis dahin einen der Wege oben nutzen und hier oben verbinden.
